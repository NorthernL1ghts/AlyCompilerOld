AlyCompiler - Building & Running on Windows (MinGW + GCC)

Prerequisites:

1. Install MSYS2:
   Download and install MSYS2 from https://www.msys2.org/.

2. Install MinGW & CMake:
   Open MSYS2 MinGW 64-bit shell (mingw64.exe), then install the necessary tools:
   pacman -Syu                    # Update package database and packages
   pacman -S mingw-w64-x86_64-gcc   # Install GCC (C compiler)
   pacman -S mingw-w64-x86_64-cmake # Install CMake

Building AlyCompiler:

1. Navigate to the Project Directory:
   cd /c/Dev/AlyCompiler  # Adjust path to your project location

2. Generate Build Files with CMake:
   cmake -S . -B bld -G "MinGW Makefiles"

3. Build the Project:
   cmake --build bld

Running AlyCompiler:

Once the project is built, run the executable:

./bld/alyc.exe


OR JUST DO THIS ---
